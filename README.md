# Projeto Site Android

Este repositório contém o desenvolvimento de um site responsivo com tema Android, utilizando HTML e CSS. O objetivo foi replicar o design e a estrutura de uma página informativa sobre a história do Android.

## Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout responsivo e efeitos visuais.

## Como Visualizar

Você pode visualizar o projeto diretamente pelo GitHub Pages [aqui](https://alexandregomesbrasil.github.io/site-android/).

## Aprendizados

Durante o desenvolvimento deste projeto, foram aprofundados conceitos de:

*   Criação de layouts responsivos (Mobile First).
*   Uso de seletores CSS avançados.
*   Organização de arquivos e pastas para projetos web.
*   Semântica HTML para melhor acessibilidade e SEO.
